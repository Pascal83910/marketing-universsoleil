**Intégration Tesla** : Le Powerwall 3 s'intègre nativement avec les véhicules Tesla, bornes Wall Connector, et potentiellement d'autres équipements Tesla.

**Onduleur intégré** : Révolutionnaire — pas besoin d'onduleur externe séparé. Les panneaux PV se branchent directement sur le Powerwall 3. Un seul équipement, un seul interlocuteur SAV, installation plus simple et moins coûteuse.

- Sensibilité aux coupures (maison secondaire, équipements médicaux, télétravail)

### 5.3 Note SAV critique — Problème connu

> ⚠️ **ALERTE SAV** : Le mode backup (alimentation du logement lors d'une coupure réseau) peut se couper après **1 à 2 minutes** malgré une charge batterie suffisante.

> **Procédure** : vérifier en priorité le câblage physique du Backup Controller, puis les paramètres logiciels du GEN24 via l'interface web. Consulter les logs de l'onduleur. Contacter le support Fronius si nécessaire.

## 6. PRODUIT 5 — CAPTEUR HYBRIDE PVT SOLINK / CONSOLAR

### 6.1 Présentation

Le **SOLINK** est un capteur hybride **PVT (Photovoltaïque + Thermique)** développé par **Consolar** (Allemagne). Il produit simultanément de l'électricité ET de la chaleur depuis un seul panneau — c'est la technologie la plus avancée du portefeuille Univers Soleil, objet de R&D active (programme CIR 2023–2025).

Univers Soleil est distributeur/installateur SOLINK en France, avec expertise de traduction de la documentation technique (DE → FR).

### 6.2 Fonctionnement

```
[Capteurs SOLINK sur toiture]
  ↓ 2 sorties simultanées :
  ├── ÉLECTRIQUE : cellules PV intégrées → ~15–18 % de rendement
  └── THERMIQUE : fluide caloporteur (eau glycolée) → ~45–55 % de rendement
  
Rendement global du capteur : 65–75 % 
(vs ~20 % pour un panneau PV seul)
```

Le capteur SOLINK remplace avantageusement :
- Les **sondes géothermiques** (coûteuses, nécessitent forage)
- Les **aérothermes** (bruyants)

Il alimente directement une **pompe à chaleur M-TEC** via le circuit hydraulique.

### 6.3 Application en mode rafraîchissement estival (R&D Univers Soleil)

En été dans le Var (T° > 30 °C), les capteurs SOLINK refroidis la nuit peuvent fournir une **source froide** pour rafraîchir le plancher chauffant sans climatisation traditionnelle. C'est le sujet de R&D principal d'Univers Soleil (CIR 3 années consécutives).

### 6.4 Schéma hydraulique SOLINK + PAC

```
[Capteurs SOLINK (4 à 16 selon projet)]
        ↓ eau glycolée
[Ballon tampon CONSOLAR SOLUS 500 à 2200 L]
        ↓
[PAC M-TEC] ← [Capteurs géothermiques si couplage mixte]
        ↓
[Plancher chauffant / Rafraîchissant]
        ↓
[Ballon ECS 300 L]
```

### 12.5 Les 9 étapes de traitement d'une objection prix

1. **Laisser parler** — ne pas interrompre
2. **Reformuler** — "Si je comprends bien, votre préoccupation est..."
3. **Valider** — "C'est une préoccupation légitime"
4. **Questionner** — "Qu'est-ce qui vous paraît cher exactement ?"
5. **Comparer** — calculer le coût de l'inaction (factures à venir sans changement)
6. **Décomposer** — montrer le reste à charge réel après aides
7. **Financer** — proposer l'éco-PTZ ou un crédit à taux compétitif
8. **Témoigner** — citer un client similaire satisfait
9. **Conclure** — "Qu'est-ce qui vous empêche de démarrer ?"
